📌 Sobre o site

Este site foi desenvolvido para simular um minimercado online, permitindo visualizar produtos, serviços e informações de contato.

🛒 Página de Produtos

- Lista de itens disponíveis no minimercado.
- Cada produto possui nome, imagem e descrição.
- Em breve: botão para adicionar ao carrinho.

🔧 Página de Serviços

- Apresenta os serviços oferecidos (ex: entregas, promoções, cartão fidelidade).
- Explicações detalhadas de como funcionam.

📞 Página de Contato

- Formulário para entrar em contato com o minimercado.
- Campos: nome, email e mensagem.
- O envio do formulário é apenas ilustrativo (não envia e-mail real).

ℹ️ Navegação

- Use o menu no topo da página para acessar rapidamente todas as seções.
- Todas as páginas possuem links entre si para facilitar a navegação.
